# STReM_source_code
This repository contains the STReM source code for a simple demonstration of 3D decomposition

Software:   Matlab 2009 or higher version

Run 'demo_cos.m' to (1). simulate a cosin shaped trajectory; (2). deconvolve the raw trajectory with 3D DH PSFs; (3). overlay the recovered trajectory with raw image.
